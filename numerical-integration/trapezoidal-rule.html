Regra do Trapzio

A Regra do Trapzio um mtodo de integrao numrica que aproxima a integral definida de uma funo ( f(x) ) no intervalo ([a, b]). Ela baseada na ideia de dividir o intervalo em pequenos subintervalos e calcular a rea de trapzios sob a curva.

Frmula

A aproximao dada por:

[ I \approx \frac{\Delta x}{2} \left[ f(x_0) + 2\sum_{i=1}^{n-1} f(x_i) + f(x_n) \right], ]

onde:

  • ( \Delta x = \frac{b-a}{n} ) o tamanho de cada subintervalo,
  • ( n ) o nmero de subintervalos,
  • ( x_0, x_1, \dots, x_n ) so os pontos que dividem o intervalo.

Passos

  1. Divida o intervalo ([a, b]) em ( n ) subintervalos de tamanho ( \Delta x ).
  2. Calcule os valores da funo ( f(x) ) nos pontos ( x_0, x_1, \dots, x_n ).
  3. Substitua os valores na frmula da Regra do Trapzio.

Exemplo de Aplicao

Vamos calcular a integral:

[ \int_0^\pi \sin(x) , dx ]

usando ( n = 6 ) subintervalos. A Regra do Trapzio divide o intervalo em pequenos trapzios, e a soma das reas fornece uma aproximao da integral.

Vantagens

  • Simples de implementar.
  • Melhor preciso do que mtodos mais simples, como a Regra do Retngulo.

Desvantagens

  • Pode ser impreciso para funes com comportamento muito varivel dentro de cada subintervalo.
  • A preciso depende do nmero de subintervalos ( n ).

Complexidade

O mtodo tem complexidade ( O(n) ), onde ( n ) o nmero de subintervalos.

Visualizao

A aproximao pode ser representada graficamente pela soma das reas dos trapzios formados entre os pontos ( (x_i, f(x_i)) ) e ( (x_{i+1}, f(x_{i+1})) ).

def trapezoidal_rule(f, a, b, n):
    """
    Aproxima a integral definida de f(x) no intervalo [a, b] usando a Regra do Trapzio.

    Parmetros:
        f (function): Funo a ser integrada.
        a (float): Limite inferior de integrao.
        b (float): Limite superior de integrao.
        n (int): Nmero de subintervalos.

    Retorna:
        float: Aproximao da integral.
    """
    x = [a + i * (b - a) / n for i in range(n + 1)]  # Pontos de partio
    y = [f(xi) for xi in x]  # Valores da funo nos pontos
    h = (b - a) / n  # Tamanho do subintervalo
    integral = (h / 2) * (y[0] + 2 * sum(y[1:-1]) + y[-1])
    return integral

import numpy as np
import matplotlib.pyplot as plt

def visualize_trapezoidal(f, a, b, n):
    """
    Visualiza a aproximao da integral usando a Regra do Trapzio.

    Parmetros:
        f (function): Funo a ser integrada.
        a (float): Limite inferior de integrao.
        b (float): Limite superior de integrao.
        n (int): Nmero de subintervalos.
    """
    x = np.linspace(a, b, n + 1)
    y = f(x)
    x_fine = np.linspace(a, b, 500)
    y_fine = f(x_fine)

    # Plot da funo original
    plt.figure(figsize=(8, 6))
    plt.plot(x_fine, y_fine, label="f(x)", color="blue")
    
    # Plot dos trapzios
    for i in range(n):
        plt.fill_between([x[i], x[i+1]], [y[i], y[i+1]], color='skyblue', alpha=0.4)

    plt.title("Regra do Trapzio: Aproximao")
    plt.xlabel("x")
    plt.ylabel("f(x)")
    plt.legend()
    plt.grid()
    plt.show()

import numpy as np

# Definir a funo sinc(x) como sin(x)/x
def sinc(x):
    return np.where(x == 0, 1, np.sin(x) / x)


# Integrao usando a Regra do Trapzio
a = 0.1  # Limite inferior (evite x=0 para evitar diviso por zero)
b = 10   # Limite superior
n = 8  # Nmero de subintervalos

# Aproximar a integral
# Calcular a integral com trapezoidal_rule
resultado = trapezoidal_rule(sinc, a, b, n)

# Exibir o resultado no console
print(f"A integral aproximada de sinc(x) no intervalo [{a}, {b}] : {resultado:.4f}")

# Visualizar o grfico com visualize_trapezoidal
visualize_trapezoidal(sinc, a, b, n)
